DBT API

Application programming interface for Dialectical Behavioral Therapy skills

Dialectial Behavioral Therapy was created by Marsha Linehan to treat acutely suicidal patients. Since its inception, many clincal studies have shown that it is very effective for a wide range of diagnoses and also for sub-clincal malaise. These skills are essential and life-changing. Skills can be accessed via module. Mindfulness, Interpersonal Relationships, Distress Tolerance, Emotion Regulation.

Dialectical Behavioral Therapy differs from other therapeutic approaches because it revolves around teaching practical, concrete skills to manage the lived reality of people who are suffering. It doesn't involve processing and talking about one's past, instead it encourages action. This API accesses a database of all DBT skills.
